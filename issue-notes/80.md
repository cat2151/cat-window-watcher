# issue cat2151のfighting-game-button-challengeリポジトリの実装を参考にして、ストリートファイター6をプレイ中は、プレイ終了まで1分間隔でチェックするループ、に入る #80
[issues #80](https://github.com/cat2151/cat-window-watcher/issues/80)

## 実装完了

cat2151/fighting-game-button-challengeリポジトリの`check_playing_game.py`の実装を参考にして、ゲームプレイ中のチェック間隔変更機能を実装しました。

### 実装内容

1. **設定ファイルの拡張**
   - `[game_playing_detection]` セクションを追加
   - ゲーム検知の有効/無効、プロセス名リスト、チェック間隔を設定可能

2. **プロセス名取得機能**
   - `WindowMonitor.get_active_window_process_name()` メソッドを追加
   - Linux, macOS, Windows で動作

3. **動的な更新間隔変更**
   - ゲーム検知時: 通常の1秒間隔 → 設定された間隔（デフォルト60秒）
   - ゲーム終了時: 通常の1秒間隔に復帰
   - 状態遷移時にログ出力

### 設定例

```toml
[game_playing_detection]
enabled = true
process_names = ["StreetFighter6.exe", "SF6.exe"]
check_interval_seconds = 60
```

### テスト

- 7つのテストケースを追加し、すべて合格
- 既存のテストも引き続き全て合格
